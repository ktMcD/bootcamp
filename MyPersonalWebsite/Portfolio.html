My most recent project is the first project in my repo that is not a school assignment. Using the skills I've gained in the Full Stack C# .NET bootcamp I'm attending, I wrote a console application that extracts text from an XPS file (similar to PDF but not as useful). The application then processes the text line by line and creates a skeleton script for my afternoon radio show. It saves me about 20 minutes because I no longer have to use several files to put together my show.
